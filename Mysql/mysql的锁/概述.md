# 锁分类

## 类型

* X: 排它锁
* S: 共享锁'
* IX: 意向排他锁
* IS: 意向共享锁

| 兼容性 | X   | S   | IX  | IS  |
| ------ | --- | --- | --- | --- |
| X      | N   | N   | N   | N   |
| S      | N   | Y   | N   | Y   |
| IX     | N   | N   | Y   | Y   |
| IS     | N   | Y   | Y   | Y   |

## 粒度

* 表级锁
* 行级锁
* Gap lock (两条记录之间的间隙)
* next-key lock (一条记录+间隙)
