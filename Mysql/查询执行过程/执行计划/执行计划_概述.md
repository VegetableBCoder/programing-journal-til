# 执行计划概述

## 各列含义

| 列名         | 含义                                                 |
| ------------ | ---------------------------------------------------- |
| id           | 一个Select语句中, 每一个select对应一个id             | \ |
| select_type  | 每个id对应的查询类型                                 |
| table        | 表名                                                 |
| partitions   | 匹配的分区信息                                       |
| type         | 针对单表的访问方法                                   |
| possible_key | 可能使用的索引                                       |
| key          | 实际使用的索引                                       |
| key_len      | 实际使用的索引长度                                   |
| ref          | 当使用索引进行等值查询时,与索引列进行匹配的对象信息  |
| rows         | 预估需要读取的记录数                                 |
| filtered     | 针对预估要读取的记录, 经过搜索条件过滤后剩下的百分比 |
| extra        | 额外信息                                             |
